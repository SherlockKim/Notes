# 一.标题

* 用法：

```text
# 一级标题
## 二级标题
### 三级标题
...
```



* 快捷键：

`ctrl` + 1  2  3  4 ...



* 效果：

> # 一级标题
>
> ## 二级标题
>
> ### 三级标题
>
> #### 四级标题





# 二.字体

## 1.加粗

* 用法：

  ```text
  **内容**
  ```

  用一对两个星号括起来      

  

* 快捷键：

  `ctrl`+`B`

  

* 效果：

  **加粗了啊**

  

## 2.斜体

* 用法：

  ```text
  *内容*
  ```

  用一对星号括起来   



* 快捷键：

  `ctrl`+`i` 

  

* 效果：

  *斜体字*



## 3.下划线

* 用法：

  ```text
  <u>内容</u>
  ```

  用一对u标签表示

  

* 快捷键

  `ctrl` + `U`



* 效果：

  <u>这是一句带下划线的</u>



## 4.脚注

* 用法：

  ```text
  [^内容]
  ```



* 效果：

  这是个[^黑莓]

  [^黑莓]: 傻逼



# 三.列表

## 1.无序列表

* 用法：

  ```
  * + 空格
  ```

  

* 效果：

  * 1
  * 2
  * 3

## 2.有序列表

* 用法：

  ```
  数字+.+空格
  ```

  

* 效果：

  1. 黑莓
  2. 白莓
     * 嵌套也行



# 四. 区块显示

* 用法：

  ```
  > + 回车或空格 
  ```



* 效果：

  > 这是个区块显示
  >
  > > 这是个嵌套的区块



# 五.代码块

* 用法：

  ```
  ​``` + 回车 + 语言（java/c/text/html/...）
  ```

  

* 效果：

  ```java
  //代码块展示
  System.out.println("fuck");
  ```

# 六.链接

* 用法：

  ```

  ```

  按住`ctrl`用鼠标点击可以跳转



* 效果：

  [百你个度](www.baidu.com)

# 八.表格

* 用法：

  ```
  |表头|表头|表头|
  |----|----|----|
  |1|2|3|
  ```

* 快捷键：

  `ctrl` + `T` 

  推荐用快捷键，不推荐用markdown语法

  

* 效果：

  | 姓名 | 性别 |
  | ---- | ---- |
  | kim  | 男   |
  | 黑莓 | 男   |



# 九. 高亮

* 用法：
